Explain normalization from DBMS notes.
High confidence 3 sources matched Re-ranked result #1

Normalization is a process of organizing database tables to reduce repeated data and improve data integrity.

  • 1NF removes repeating groups.
  • 2NF removes partial dependency.
  • 3NF removes transitive dependency.

Simpler version: Normalization means splitting messy tables into cleaner smaller tables so the same data is not repeated again and again.
